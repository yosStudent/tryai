# ASCII Games Collection 🎮

A retro-styled game collection featuring three classic games with amazing ASCII art aesthetics!

## 🎲 Games Included

### 🏓 Ping Pong
Classic two-player paddle game. Control your paddle with arrow keys or W/S, and don't let the ball pass! First to 10 points wins!

### 🐍 Snake
The timeless snake game. Eat food to grow longer, but don't hit yourself or the walls! Features high score tracking.

### 🎯 Same Game
A colorful tile-matching puzzle game. Click on groups of 2+ same-colored tiles to remove them. Larger groups earn more points! Clear the entire board for a bonus!

## 🎨 Features

- **Retro ASCII Art Theme**: Gorgeous glowing green ASCII art headers and decorations
- **Fully Playable Games**: All three games are fully functional with smooth gameplay
- **Score Tracking**: High scores saved locally for Snake and Same Game
- **Responsive Design**: Works on desktop browsers
- **No Dependencies**: Pure HTML5, CSS3, and vanilla JavaScript

## 🚀 Getting Started

Simply open `index.html` in your web browser to start playing!

Or run a local web server:
```bash
python3 -m http.server 8000
```

Then navigate to `http://localhost:8000` in your browser.

## 🎮 Controls

### Ping Pong
- **Arrow Keys ↑↓** or **W/S**: Move your paddle
- **Space**: Start/Pause game

### Snake
- **Arrow Keys ←↑→↓** or **WASD**: Change direction
- **Space**: Start/Restart game

### Same Game
- **Mouse Click**: Select and remove groups of tiles
- **NEW GAME Button**: Start a new game
